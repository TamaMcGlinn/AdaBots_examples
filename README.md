AdaBots code editor
===================

Welcome to the AdaBots code editor!

Press control-shift-B to build, control-shift-\` to open a terminal and run the program from the ./bin directory, for example: ./bin/example

Make sure your bot in minetest is listening to localhost port 7112 (or some other port in the range 7112 - 7119 if you pass it in Create_Turtle.
